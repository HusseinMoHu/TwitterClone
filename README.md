# Twitter Clone

> Twitter platform built with Nodejs, Express, JavaScript, MongoDB, Mongoose.

## Usage

### Env Variables

Create a .env file in then root and add the following

```
NODE_ENV = development
PORT = 8080
MONGO_URI = your mongodb uri
```

### Install Dependencies

```
npm install
```

### Run

```
npm start
```
or

```
npm run dev
```
